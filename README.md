# EncryptionAppKmm
## General info
This is a training project that aims to use kmm in projects with the aim of first experimenting and discovering the limits of this technology
## Technologies
Project is created with:
* kotlin : 1.6.10
* swiftUi : ###
* Kmm plugin : 0.3.0
* Kto: 1.6.7
* serialization : 1.6.7
* sqldelight : 1.5.3
* back4app servier side 
## Setup
To use this project with another Apis you can use back4app server side or any server side provide same data schema
